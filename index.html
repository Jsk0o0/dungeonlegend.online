updated 2026·05·13

  1. 2026·05·13
    AURA

    突破灵气环身 · 六色试演

    突破金身、渡劫紫雷、狂暴血焰、灵泉治愈、白虹飞升、魔化黑曜六款灵气随心切换;周身条纹涌动、菲涅尔光勾轮廓、星点闪烁脉动呼吸;角色一动空气被压扭折射、颈口符文环旋,远看就是一团活的灵压。

    ELITE

    精英怪物登场

    在编辑器里勾选即可让任何刷怪点升格为精英:战斗属性双倍、体型放大五成、头顶挂"精英·"标识、身上一圈粉色灵气随行;玩家远远一眼分得清是普通杂兵还是这片地图的硬骨头。

    MOBS

    怪物动作彻底校准

    老版本怪物每打五下会突然倒地一次再爬起的诡异表现彻底修复;受击、技能、死亡的姿态分槽对位,不再混淆;空动作的怪物也不会再原地呆立。

    DUNGEON

    切层即断仇

    副本中跨楼层传送,上一层未结清的怪物会立即脱离锁定:再也不会出现"换了层之后血条还在掉"的隔空追击。

    SKILL

    环形技能定位锁脚下

    冰环、十字斩这类围绕自身的范围技,右键释放时圆心稳稳锁在角色脚下,不再被鼠标方向带偏到错位;左键仍可远程精确放置陷阱型圆环。

    MAP

    大地图与小地图焕新

    按 M 立刻看到当前区域的真实俯视图,小地图也跟着换上同一张底图;切换地图时即时烘焙完成,不再有按下后再渲染的短暂卡顿,角色走到哪图就跟到哪。

  2. 2026·05·12
    SKILL

    技能手感全面打磨

    陨石、火炎弹、玄冰弹、雷电斩、冰环全部正确造成伤害；伤害数字与命中火花在飞行特效真正击中目标的一瞬同时弹出；右键技能直接落到鼠标所指位置,告别二次选位。

    DUNGEON

    副本跨层连贯推进

    同一副本切换楼层不再播完整入场动画,角色直接以战斗准备姿态接上一层节奏;不同副本之间仍保留壮丽入场,层次感分明。

  3. 2026·05·11
    MOBS

    怪物战斗反应更利落

    挨打不再发呆,被攻击下一秒就反扑;玩家躲到障碍后也会持续锁定,绕回来继续打;选中圈细如发丝,命中火花贴着身体而非飘在头顶。

    WORLD

    地图精度与可玩区域校准

    墙体、桥栏、装饰物不再被误判可行走,角色不会卡在虚空一角;每张副本逐层独立校对地形,刷怪点自动避开墙体。

  4. 2026·05·10
    AI · TERRAIN

    怪物寻路全面重构 · 地形精度还原

    穿越传送门后仍能精准追击、墙角自动绕行、被攻击坚定锁仇直至击杀；地图行走数据按设计师手稿一比一还原，可走区域翻倍，告别站桩挨打与寻路死锁。

    RANGED · COMBAT

    远程武器系统 · 弓箭弹道

    弓接入：左手持弓姿势、独立旋转校准；攻击距离 8 米触发，箭矢从弓位发出追踪目标，飞行中目标移动也能命中；模型轴向自动校正，飞行姿态稳定。

  5. 2026·05·09
    UX · POLISH

    装备背包交互打磨 · 物品拖拽全链路

    消耗品在快捷栏 ↔ 背包之间双向拖拽，落点自动切到匹配分类标签并选空位；幽灵格容错避免越界；服务端实时同步槽位，下线重登位置一致。

  6. 2026·05·08
    COMBAT · ASSET

    四类资源体系完整复刻

    动画 · 音效 · 元素法阵全打通：6 种属性吟唱阵、抛物线击倒、男女语音独立资产、女性动画自动并入共享动画库。

  7. 2026·05·07
    SKILL · STORAGE

    技能形态重构 + LMB/RMB 服务端持久化

    13 种 skill_form 完整覆盖；火球加跟随光、命中爆炸、击退；LMB/RMB 槽位按角色 ID 在服务端持久化。

  8. 2026·05·06
    UI · CORE

    黑金 UI 重塑 + 性别系统 + 共享动画 + 技能释放链路

    主题统一为修仙黑金；男女角色独立 voice；技能从 hover/lock 到吟唱到飞行的完整链路一次落地。

  9. 2026·04·30
    BRAND · POLISH

    落地页上线 + 宠物蛋掉落 + 战斗打磨

    修仙地牢传说官方品牌页发布；战利品系统加蛋类孵化；怪物攻击动画与拾取交互全面打磨。

  10. 2026·04·28
    MILESTONE

    项目初始提交 · 倒计时归零

    客户端 + 服务端双栈搭建完成。从此每一行代码都是修仙路上的一阶台阶。
